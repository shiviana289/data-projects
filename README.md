# Energy & Climate Analytics Projects

Projects from my Applied Analytics minor and GIS coursework at USC, 
focused on energy systems, physical risk, and infrastructure data.

## Projects

**ARIMA Energy Demand Forecasting** (`Project_3_ARIMA_.ipynb`)  
Time-series modeling on 10-minute interval power consumption data from 
Tetouan, Morocco. Includes stationarity testing, differencing, ACF/PACF 
analysis, and ARIMA model evaluation using MAPE and RMSE.

**Wind Generation Forecasting** (`WIND_Project_3_ARIMA_.ipynb`)  
Same ARIMA methodology applied to a wind generation dataset — directly 
relevant to utility-scale renewable asset forecasting.

**AutoARIMA — Energy Consumption** (`Project4_AutoARIMA.ipynb`)  
Automated model selection using pmdarima on the same power consumption 
dataset. Compares AutoARIMA against manually tuned ARIMA models.

**NYC Building Emissions Pipeline — Local Law 97** (`NYC_LL97_Building_Emissions_Pipeline.ipynb`)  
End-to-end data pipeline ingesting NYC building energy data via the NYC 
Open Data API. Feature engineering across 300+ fields, modular cleaning 
functions, and compliance-focused analytics for LL97 emissions limits.

**Customer Segmentation — ML Classification** (`Project_5_MachineLearning.ipynb`)  
Supervised classification using logistic regression and scikit-learn 
pipelines. Includes train/test split, StandardScaler, GridSearchCV, and 
evaluation metrics.
